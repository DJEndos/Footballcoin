How it works

TheSportsDB uses the public key "3" which works from the browser (CORS enabled). It's great for demos but has limited real-time coverage.

Football-data.org and livescore-api.com require an API key and a backend proxy (they block direct browser calls on free plans). Point FCLiveScore.fetchFromProvider in js/livescore.js to your proxy endpoint.

The Fetch Now button on the Live Matches tab lets you pull data on demand without waiting for the auto-sync interval.
